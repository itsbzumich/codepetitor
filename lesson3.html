Lesson 3

 

Arithmetic

 

Introduction to Arithmetic

What if you were told a computer could do math? Well, I don’t think you are that surprised. Computers are extremely smart. Of course they can do math! In Java, we can do addition, subtraction, multiplication, division, and a ton of other arithmetic functions. Let’s focus on the basics first.

Addition, use the "+" sign. 1+4=5.
Subtraction, use the "-" sign, 3-2=1.
Multiplication, use the "*" sign. 5*4=20.
Division, use the "/" sign. 6/3=2.

How are we going to use these in a code? One thing we can do is print it out directly, as follows.

System.out.println(1+3);
System.out.println(5*7);

As we can see here, our statements print out 4 and 35 to the system. One thing to note is that for numbers, we don’t need quotes. Look what happens when we use quotes.

System.out.println(“1+3”);

It will literally print "1+3", not 4. So be careful!

Other Arithmetic Operations

We can do a bunch of other cool math in Java. However, the way we do it is a bit different that simple stuff like addition. We can all some special operations when we type in “Math.function_name(function_parameters);”. We replace function_name with the name of the function. We replace function_parameters with the numbers we want to apply the function to. If you don’t understand, don’t worry. Just copy what I gave you! That’s the best way to learn!

These are the three functions we will be learning today.
Math.max(a,b); This returns the bigger value between a and b.
Math.min(a,b); This returns the smaller value between a and b.
Math.pow(a,b); This returns a to the power of b. Returns a double value.

It’s that simple! All we must do is call these functions to return the correct value. Let’s try printing out a few of these operations.

System.out.println(Math.max(10,5)); This will print 10, which is greater than 5.
System.out.println(Math.max(11,4)); This will print 11, which is greater than 4.
System.out.println(Math.min(12,4)); This will print 4, which is less than 12.
System.out.println(Math.min(7,7)); This will print 7, since it's the smallest/only number.
System.out.println(Math.pow(5,3)); This will print 125, since 5*5*5=125.

Math with Variables

Instead of using numbers like we did previously, we can use variables we created earlier in the program. Assume we created integers x and y earlier. We can add, subtact, etc.. these two variables to get the value.

int x=10;
int y=11;
System.out.println(x+y);
System.out.println(x*y);
System.out.println(x);
int z=x+y;

Let's go over what this code does. First two lines create variables x and y, which we set to 10 and 11. We then print out x+y, which is 21. Printing out x*y, we get 110. Finally, we can also directly print out the value of a variable, as we did in line 5. This prints 10, the value of x. If we do System.out.println("x"), we will end up printing the character x, which is not what we want. Finally, we can also create a new variable and set it to equal some combination of other variables. In this case, we have z=x+y, which results z being equal to 21.

Special Topic

Let’s look at one more concept before we end the lesson. Check out the following code down below.
int x=10;
int y=6;
x/y=?
 Is the answer 1.6666666...? No! In Java, x/y=1. When we divide 2 integers by each other, we get the answer without the remainder. Be sure to keep this in mind.

How do we find the remainder? Use %. This will give us the remainder when we divide two integers by each other.
10%3=1
11%4=3
19%5=4
 What if we want the decimal? As we mentioned previously, use the double variable type for decimal numbers. Java won’t allow us to produce decimals from integers. Therefore we have two options.

Option 1: Use Doubles

This will allow us to have decimals outputs. This is because the double variable type was designed to be able to contain decimals. See the example down below.
double a=10;
double b=3;
System.out.println(a/b);
This code will print 3.33333333…

Option 2: Cast the Integers to Doubles (harder)

We can also cast our integers to doubles. This means that when we do our division, we want to specify that we want the integers to act as doubles for just the expression. In other words, we are telling the computer that for this line of code, our ints are now doubles, allowing the computer to output a value with decimals. We do this by writing (double) next to the variable we want to cast as a double.
int a=10;
int b=10;
System.out.println((double)a/(double)b);
This code will print 3.33333333… a and b are still integers.
