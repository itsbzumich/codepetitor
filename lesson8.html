Lesson 8

 

Problems(Medium)

Round 2

Time for some harder problems! Same drill as last time. GOOD LUCK!


1. Given an int array arr, return the number of 7s in the arr.

2. Given two int arrays a and b. Create a new array with a size of 2, with the first element being the first element of a and the second element being the first element in b.

3. Given a number n, return true if it's a multiple of 4 or 7 but not both.

4. Given an array of doubles, arr, find the max value of an element of the array squared and return it.
Be careful! -7 squared is bigger than 6 squared even though -7 is smaller than six.
In other words, square each element and find the largest one.

5. Given three ints a, b, and c, return the sum of unique integers. This means that if a number appears more than only 1 time, it only counts once.
EX: a=1, b=1, c=2. It's 3 since 1 can only count once.
