Variables

Introduction to Variables

In algebra, we use variables to represent a value. By referring to the variable, we refer to the value. So with the equation x+1=2, x represents a number, in this case 1. We typically use a variable to represent something we are unsure about.

In computer science, variables are used the same way. They represent something, typically something we need to change or manipulate. Unlike in math, we typically know the value of a variable at the start. Therefore, when we create a variable, we usually set it to a value.

To start, let’s focus on numbers, in particular, whole numbers. To make a number variable, use the following format.

int a=1;

“int” is the type of variable. You’ll soon learn about the other things variables can represent besides numbers! In this case, however, we want an integer(whole number), so we put the “int”. “a” is the name of the variable. Like in math, all variables have names, so we must name our variable. Finally, we use “=1” to set the value. Like all lines of code, we end with a semicolon “;”.

Now, the computer knows that “a” is an integer with a value one. It will remember this entire code unless we want to change it, which we will discuss later.Another way to do this is as follows.

int a;
a=1;

First, we declare an integer a. It doesn’t have a value yet. By not assinging a value, we created a variable that we can use for the future. Then, we set it to 1 on a separate line. Let’s look at another example of code.

int a=1;
a=2;

We’ve already seen what the first line does. It creates an integer variable "a" with a value of 1. Line 2 simply changes the value of the variable. So in this case, "a" changes from 1 to 2.

Types of Variables

Variable Type Declaration What is it?
Integer int a=1; whole numbers
String string s="yes" words, characters, and sentences
double double s=1.2 any number(whole and decimal)
boolean boolean a=true; true or false

Now, let’s talk about other types of data you can store in variables. The table above shows a great summary of them! I'll provide some more details about them.

A double is the same as a decimal number. Therefore, if you want to make a variable and set it to a decimal value, use a double. It can also represent a whole number. You usually want to use "int" for whole numbers. Here's another example of us declaring a double.

Double a=1.234;

Strings can be any size list of characters. Think of them as words, phrases, or a random list of characters put together. Use “ " or double quotes around them.

string word= "apple”;
string goat= "I am the GOAT at coding yessir!”;

GOAT=Greatest of All Time
