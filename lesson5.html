Arrays

 

Introduction to Arrays

Many times, we want to store multiple values instead of just one. That’s when using an array comes in handy. This allows us to store many many numbers, strings, or booleans instead of having to make one at a time. All of these elements are stored in the same array! Let’s give an example of an array and how you make one!

int[] arr=new int[10];
 In the line of code above, we are creating an array of 10 integers. The name of the array is called arr.

The table above represents arr. It’s simply a java type that can store 10 integers. Currently, nothing’s in it because we never set it to equal anything. We will show you how to set elements, or spaces in the picture, to equal certain values. Keep in mind arr is a variable name that refers to the array. Also, the size of this array is fixed. We can't change the size once it's declared.

Indexing and Accessing

To access each element in the array, we must use a special call. As you can see in the drawing of the array from the previous section, I labeled each box with a number underneath. This will represent the index of the array. In an array, the first element has an index of 0. This may sound weird. Why isn’t the first element index 1? Well, computers are weird and I couldn’t tell you. But from there, we count by one each time, so the second element is index 1, the third element is index 2, and so on. So if we wanted to see the 5th element in the array, we would look for index 4.

Now comes the question of how to change/view the value of an element of an array. Let's check out the code down below.
arr[0]=1;
 This changes the value of the 0th index, or the 1st element, to 1. Be careful. Don't try to change an element that doesn't exist. If the size of the array is 10, don't try to access index 10 or anything higher.

If we want to print the value at a certain index, we can do so by calling print(arr[index]); No quotes here either!
arr[4]=10;
System.out.println(arr[4]);
 This will print out 10, the value of arr[4].

Extra Topics

While we focused on int arrays for the first few, arrays can be of any type. int, double, boolean, and string all work! I put some examples down below.

We can also create arrays as follows.
int[] j={1,3,4,5};
This basically allows us to set all the elements at the start instead of going one by one. Here, we directly write out the elements of the array. It’s efficient if we already know what we want to put inside the array. Be sure to study how we did this so you can replicate it.

Let’s continue with the previous array j we just created. We can set elements of the array equal to the values of the other elements. j[3]=j[1]+j[2];
 This sets j[3] equal to 7, as j[1]+j[2]=7.

Finally, I'm going to cover the size of an array. Let’s say you forget the size, or don’t know the size of an array. To get the size, call arr.length. Repalce arr with the name of the array.

For example:
int[] random=new int[4];
int size=random.length;
 size is now equal to 4, as 4 is the size of random.
