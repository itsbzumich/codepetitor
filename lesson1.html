We are going to learn how to write our first line of Code! We will be using the language Java. At the end of the lesson, you will win a Frozen Sticker.
Let’s start with the famous Hello World code. It shows how you print something. In this case, it’s our first code, so we want our computer to say hello to the world! The code is shown below. It prints out the words "Hello World"

public static void main (string args[]){
System.out.println(“Hello World”);
}

When I first saw this code, I could kind of guess how it worked. One thing I was never sure of was what “public static void main (string args[])” meant. Don’t worry! For now, let’s call it the “Main Header”. We call it the Main Header because it’s where you write the main code. Don’t worry about the main header for this course. I want to introduce it to you so you aren’t confused about it in the future.

After the header, we must put brackets, { }. After we write the main header, the brackets are where we put the code inside. Make sure you close the bracket. We can’t have a “{“ with no “}”.

Then, we can write our print command. System.out.println(“Hello World”); is what prints out “Hello World”. System.out.println means we are printing to the system. The “ln” at the end of print means that we will move to the next line for our next print statement. Don’t worry about it too much.

We then put () to indicate what we want to print, and write what we want to print inside the (). Therefore, we write “Hello World” within the ().

The semicolon is there after every line of java. Don’t use it after the header. Sorry about talking about the header so much. But I when I was learning coding, I was really confused because of it. So I’m trying to make sure you guys aren’t as confused as me. For now, make sure that after you print something, you put a semicolon afterward.

Be exact! Don’t forget anything. Make sure what’s in the “ " matches exactly what you want to print. Let’s practice!
